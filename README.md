# Hei Si

_✨ NoneBot2 黑丝 （备考中，可能会更新）✨_

  **由于jsd在国内无法访问的问题，国内服务器或无法使用，待后续更新**

</div>


用于发送色色图片，内置CD

请在 `.env.dev`中做出如下配置：
```

heisi_group=["237161342"] #黑丝白名单
heisi_cd=30 #黑丝cd时间（s）

```

## 安装💿
`pip install nonebot-plugin-heisi`


## 导入📲
在**bot.py** 导入，语句：
`nonebot.load_plugin("nonebot_plugin_heisi")`



## 指令💻
```
.黑丝
.丝袜
```
